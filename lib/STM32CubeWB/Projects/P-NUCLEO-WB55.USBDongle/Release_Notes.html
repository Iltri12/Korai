Purpose

This directory contains the examples and applications to demonstrate the capabilities of the P-NUCLEO-WB55 Kit.

Most of those projects are generated with STM32CubeMX tool to initialize the system, peripherals and middleware stacks.

Several applications are provided ready to use with IAR, KEIL and SW4STM32 to demonstrate the Bluetooth Low Energy, Thread and Mac 802-15-4 capabilities of the device.

A particular attention must be taken when using BLE, Thread or Mac 802-15-4 as a specific binary must be loaded inside device memory for each application. You can refer to the readme.txt of each application for the detailed information and usage.

They can be found under the following directories:

  • Projects\NUCLEO-WB55.USBDongle\Applications\BLE
  • Projects\NUCLEO-WB55.USBDongle\Applications\Thread
  • Projects\NUCLEO-WB55.USBDongle\Applications\Zigbee
